Project System Blueprint

Login Page

This is the Login Page wherein the User (Admin/Employee) can login their account. There are 2 ways of logging an account. First, the user can manually input his/her Username and Password Account, then, Second, the user can login his/her account using only his/her unique QR Code.

Dashboard

In this interface, only the Admin Users can access this interface. It shows the summary of the departments, shifts, employees, and how many employee users are active. In the left side of the interface you see there the different interface of the system.

Department Interface

In the department interface you can see here the different department of the company. Also, the user can also edit and delete any of the departments that can see in the Action part and aside from that the user can also create new department, simply just click the "Add New Department" and input any information about it then click "Add".

Shift Interface

In this part, shows the time shift of the company. In the company there are 2 types of time shift. The first shift is from 8:00 am to 7:00 pm and the second shift is from 8:00am to 5:00pm. The same with the department interface, this interface can also edit and delete the shift time and also can add new time shift.

Employee Interface

In this interface, it shows the details or information of the employee such as the Name, Shift, Gender, Profile Image, Date of Birth, and lastly the hired date. Additionally, if there is missing or wrong information about the employee, the user can edit and delete the employee information if the employee was out in the company. If there is new employee, simply click the "Add new Employee".

Adding New Employee

After clicking the "Add new Employee", the system will go automatically to this interface. The admin user can input the data or information of the new employee. After the data input of the new employee, the admin user can click the "Add to System" to record the information of the new employee in the system.

Location Interface

In the location interface shows the different location of the company. For example, the company has different offices, therefore, the system will record the location of the offices. Again, the admin user can edit, delete, and create new location of the offices.

Report Interface

In this interface, shows the report of every employee if they have something to clarify about their time records. It has a search bar in the top right corner to easily search the reports of a specific employee just input the date and the department of the employee.

Employee Attendance Report

This is the part of the report wherein, if the employee wants to see his/her record and need some clarifications about it. The admin user can print the employee record to show it to the specific employee if there is something to clarify.

Attendance Form

This is the part where the employee can input his/her time of login. You can see in the interface that you can put image there, that is where the QR of the employee should put then click the "Turn it" button. After clicking the button the system will message to the user.

QR Scan Sample

This is the part where the employee can input his/her time of login. You can see in the interface that you can put image there, that is where the QR of the employee should put then click the "Turn it" button. After clicking the button the system will message to the user.
